## PRD: [n8n 실전 프로젝트 5] 명함 인식 및 DB 자동 등록 앱

### 1. 🎯 개요 (Overview)

* **프로젝트명:** [n8n 실전 프로젝트 5] 명함 인식 및 DB 자동 등록 앱
* **요약:** Flutter로 개발된 모바일 앱으로, Upstage OCR API를 사용해 명함을 인식하고, n8n 워크플로우로 데이터를 전송하여 DB에 자동 등록하는 교육용 프로젝트입니다.
* **문제 정의:**
    * 수동으로 명함 정보를 입력하는 것은 비효율적이며 오류가 발생하기 쉽습니다.
    * 수강생들은 모바일 앱(Flutter)에서 외부 API(OCR)를 연동하고, 이 데이터를 백엔드(n8n) 워크플로우와 연결하는 실습 경험이 필요합니다.
* **솔루션:**
    * Flutter 앱을 통해 명함을 스캔/업로드합니다.
    * Upstage OCR API로 명함 텍스트를 추출합니다.
    * 추출된 데이터를 n8n 백엔드로 전송하여 DB에 저장하고, 텔레그램으로 완료 알림을 받습니다.
* **핵심 목표 (교육적 목적):**
    * 수강생이 웹, 백엔드 경험 후 모바일(Flutter) 개발에 입문하도록 합니다.
    * 외부 상용 OCR API(Upstage)의 연동 및 사용법을 학습합니다.
    * 모바일 앱(Client)과 n8n 워크플로우(Server) 간의 데이터 연동 과정을 실습합니다.

### 2. 👩‍💼 사용자 (User)

* **학습 대상 (Target Audience):**
    * 개발 능력을 확장하고 싶은 개발자
    * 개발적인 백그라운드가 부족하다고 느낀 개발 초급자
    * AI를 이용한 개발 능력을 가지고 싶은 기존 개발자
* **사용자 스토리 (User Stories):**
    * 나는 **(수강생은)** 앱의 홈 화면에서 **카메라로 명함을 촬영**하거나 **갤러리에서 이미지를 선택**할 수 있다.
    * 나는 **(수강생은)** 업로드된 명함 이미지에서 **인식된 텍스트** (이름, 이메일, 전화, 회사명, 직책, 홈페이지, 주소)를 **폼(Form) 형태**로 확인할 수 있다.
    * 나는 **(수강생은)** OCR이 부정확하게 인식한 내용을 폼에서 **직접 수정**할 수 있다.
    * 나는 **(수강생은)** **'전송' 버튼**을 눌러 최종 명함 정보를 **n8n 워크플로우로 전송**할 수 있다.
    * 나는 **(수강생은)** 명함 정보가 n8n에 의해 성공적으로 처리되면 **텔레그램으로 알림**을 받고 싶다.

### 3. 📱 제품 기능 (Features)

#### 3.1. 핵심 기능 (MVP - Minimum Viable Product)

* **1. 홈 화면:**
    * 명함 촬영을 위한 [카메라 버튼]
    * 갤러리에서 명함 이미지를 불러오기 위한 [사진 업로드 버튼]
* **2. 명함 인식 및 편집 화면:**
    * 촬영/업로드된 이미지를 `Upstage OCR API`로 전송하여 텍스트 인식
    * 인식된 내용을 아래 필드로 분류하여 수정 가능한 폼에 표시
        * 필드: 이름, 이메일, 전화, 회사명, 직책, 홈페이지, 주소
    * 수정된 폼 데이터를 n8n 백엔드(웹훅)로 전송하는 [전송 버튼]
* **3. n8n 백엔드 워크플로우:**
    * (n8n) Flutter 앱으로부터 명함 데이터(JSON) 수신
    * (n8n) 수신된 명함 인식 내용을 DB에 저장
    * (n8n) 처리 완료 시, 텔레그램을 통해 "처리 완료" 알림 전송

#### 3.2. 제외되는 기능 (Out of Scope)

* **앱 내 명함 관리 기능:** 목록, 검색, 상세 보기, 삭제 기능은 구현하지 않음. (목적: 교보재 단순화)
* **사용자 계정 및 로그인:** 별도 인증 절차 없음.

### 5. 🎨 디자인 및 UX (Design & UX)

* **UX/UI 참조:** 디자인 참조 화면 첨부함.
* **핵심 사용자 플로우 (User Flow):**
    1.  [앱 실행] → [홈 화면]
    2.  [카메라 촬영] 또는 [사진 업로드] 선택
    3.  (OCR API 연동) → [인식/편집 화면] (인식된 내용이 폼에 채워짐)
    4.  (선택) 사용자 폼 내용 수정
    5.  [전송 버튼] 클릭 → (n8n 워크플로우로 데이터 전송)
    6.  (n8n) → DB 저장 및 텔레그램 알림 발송
    7.  (App) → "전송 완료" 피드백 표시 후 홈 화면으로 복귀
