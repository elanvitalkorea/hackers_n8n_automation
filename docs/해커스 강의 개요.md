
### 해커스 강의 개요
- 협업대상 : 해커스HRD
	- 기업 업무 교육을 담당해왔으며 AI 교육도 확장중에 있음
	- AI 교육을 크게 확장하기 위해서 AI 전문가인 나에게 컨택해옴 
	- B2C, B2B 통합 사이트 2개월 이내 OPEN 예정 
- 과제명 : 20년차 개발자가 알려주는 AI  X n8n 업무자동화 실전프로젝트
- 과정목표
	1. 개발의 필수 개념 공부를 통해 스스로 확장이 가능하도록 한다 
	2. 업무 자동화의 개념을 이해시키도록 직접 경험한다
	3. Backend 직접 구현을 통해서 개발에 대한 깊은 이해가 될수 있도록 한다
- 학습대상
	1. 바이브코딩으로 개발 능력을 확장하고 싶은 개발자
	2. 바이브코딩을 경험했지만, 실제 개발적인 백그라운드가 부족하다고 느낀 개발 초급자
	3. 전통 개발은 잘하지만 AI를 이용한 개발 능력을 가지고 싶은 개발자
- 자사 과정만의 차별성
	1. 코딩의 개념을 제대로 이해하고 활용할 수 있다
	2. Frontend 위주의 바이브코딩과 다르게 Full Stack 개발 경험이 가능하다
	3. 업무자동화의 실제 적용할 수 있는 방법론을 가르쳐준다
